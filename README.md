
## Description
一个nestjs的todo项目

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development (watch mode)
$ npm run dev

# development
$ npm run start

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
